Guile Scheme Overview

20.04.2022 03:00
Guile Scheme is actively maintained scheme dialect by the GNU project and has receieved continued improvement over the years. Most recently, version 3 introduced JIT compilation to speed it’s performance considerably. Like (some) other Lisps, allows you to interact with and modify your program while it’s running. It also has good C interop for both C FFI or for embedding in (and adding interactivity to) C programs. Overview of Common Lisp

04.04.2022 14:15
In this post I’ll give my thoughts on the strengths and weaknesses of programming in Common Lisp. CL is particularly well suited for building applications which can be inspected and modified as they run. Applications can be running locally, on a server, or even on embedded devices. CL’s macros empower you to cut down substantially on boilerplate and create useful DSLs, which can sound very abstract, but in practise it just means that you get the ability to shape the language to the problem you’re working on far more than you can with most other languages.
